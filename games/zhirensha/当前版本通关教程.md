# 《纸人煞》当前版本通关教程

> 含全部剧透、密码、开发者工具线索与隐藏操作。本文对应“花瓣式叙事重构”后的版本。

## 游玩前

- 建议使用桌面浏览器，并允许页面使用 `localStorage`。跨章遗物、道藏和部分结局回声依赖它保存。
- 旧物补卷在 `secret/recovered-ledger.html`：B-037、C-012、D-007、E-006 分别对应第二至第五章的档案校验。
- 游戏内不会再显示“下一步”。卡住时可查看页面源代码、选中文字、悬停暗处、右键场景空白处，或回到前一场景比对物件描述。

## 第一章：纸人煞

1. 阅读“风水旧稿”和“合婚帖”后，从“关于”页的极小入口进入里世界。
2. 罗盘正确顺序为：**木 -> 火 -> 土 -> 金 -> 水**。
   - 正路在旧稿中以浅墨留下；罗盘空白处右键会出现“生之序，始于青”。
3. 符纸的真实数字只在“凝神”约 **70% 至 73%** 的狭窄区间显影，为 **3704**。朱砂横线穿过的是 `0`，不是 `8`。
4. 四格数字填完后等待约三秒，正确会自动进入木柜。
5. 木柜密码为 **5791**。它取合婚帖的日、时柱换算：戊午、壬子 -> `5 7 9 1`。

### 第一章深度探索

- 木柜与书房边缘藏有陈氏宗祠钥匙；查看三块牌位背面，得到陈家血脉诅咒图。
- 妆奁镜背可读“往昔之匣”：婚书的证婚人是归元子，另有终南山旧照与合心佩。
- 书房抽屉的药方和病亡册解释陈守一签契的根源。
- 收齐七种纸人小样，尤其留意“复仇纸人”的点睛禁忌。

## 第二章：阴司债

### 进入频道

进入页面只会看见终端与光标。密码是 **xiaoyue**：

1. 首页源代码有“钥匙碎片1”，说明取她的名字并用小写拼音。
2. 在“关于”页选中并复制极小文字“点我的人，已入局”，屏幕会短暂出现 **xiao**。
3. “风水旧稿”正文有一段接近底色的选中文本：“月的拼音，是钥匙的最后一块。”即 **yue**。

输入错误时终端只会轻微抖动并清空。

### 抗辩主线

1. 阅读 B-037 补卷后，在契约墨污上按住鼠标反复涂抹约十次，显出代签手印；这会登记涂改证据。
2. 在匠人谱系查看陈月娘，取得病历，确认签约不是唯一救命办法。
3. 在役鬼工坊修复叁柒符纸，或在回音壁比对旧戏日期，取得现场证词。
4. 契约页的酆都地图，第七殿位置有极小墨点。进入陆斩密室后，在孽镜上快速叩击三次，打开善魂堂，取得被划掉的“从轻发落”条文。
5. 残魂区角落的旧药包关联赵婶，她证明苏婉临终仍惦念陈五行。
6. 集齐证据后进入归档。帮助青衣、让叁柒保有自主性，会让判词倾向“阴阳两消”。

## 第三章：忘川渡·昆仑篇

### 枢纽与主线

三生石是本章枢纽。渡口簿、伞影回廊、瑶池废墟、九重塔和织缘工坊均可任意顺序进入。

昆仑不是由九层封印直接打开，而是由三生石前的三件青绡遗物开启：

| 遗物 | 所在区域 | 取得方式 |
| --- | --- | --- |
| 玉簪 | 瑶池废墟 | 收拢昆仑玉屑后，从枯井打捞。 |
| 青丝 | 九重封印塔 | 在塔内寻找一缕极细青色线头。 |
| 旧帕 | 织缘工坊/伞影回廊 | 在红线旁发现同心结留下的帕角。 |

三件遗物在三生石前同时亮起后，石面裂出通向昆仑的白线。

### 各区要点

- **瑶池废墟**：这里集中青绡与羿清初见、西王母私心、蟠桃四季与枯井线索。春季长按花瓣可得玉兔玉佩；秋季叶脉为 `0917`；冬景融冰后可见旧日约期。连续触碰枯树七次可听见西王母自白。
- **九重封印塔**：月桂顺序为 **玉兔 -> 蟾蜍 -> 桂树 -> 斧头**；弱水暗数为 **4**；瑶池缺席者是 **何仙姑**；羿清身份需由化仙池玉佩确认。塔中皮影回响集中呈现青绡的千世情劫。
- **织缘工坊**：在红线节点按 `2 -> 0 -> 1 -> 2` 的区域顺序试编，可得到记忆丝线。这里的旧帕可带去第四章。
- **三生石**：搜索苏婉、再搜索陈五行并持续按 Enter，可见两世擦肩的无声皮影。
- **旧伞铺**：收集五片残片后，老何说出“蟠桃核里有毒”。油纸伞转到约 **217 度**，会露出归墟方向。

## 第四章：归墟·烛龙

### 双钥匙结构

归墟之门与观因果台是中心。因果回廊、司命台、星落池和烛龙心室可自由来回。

烛龙心室需要两把钥匙同时归位：

1. **司命台**：用画布端点理顺三条主线，分别对应阴司、昆仑、凡尘。
2. **星落池**：把三段残句拖到对应的池底位置，拼回三种结局记忆。

两项都完成后，再去心室。此时把第三章旧帕拖到竖瞳附近，会出现额外的私人提问。

### 归墟子与烛龙

- 归墟子处停留五秒可听见纺线低语；连续点击他三次，会掉落第一片玄澈记忆。
- 门环双击、因果树长时间悬停、右键因果树、`Ctrl + Shift + X`，可补齐其余记忆。五片合成《玄澈手记》后，末句会提到素心的回信。
- 烛龙心室中可保持十秒沉默，观看其孤独回忆；也可以用问心镜与旧帕触发额外对话。
- 最终归档时，先保证纺线、星池和问心都完成。结局由你在问心中的态度、是否制造死结、自观玉与沉默记录共同决定。

## 第五章：道劫·八卦归元

### 自由进入六门

太极图是枢纽。归元子唯一明言是：“**卦分先后，顺天而行，水始火终。**”

六门可任意进入：坎、坤、震、巽、乾、离。按 **坎 -> 坤 -> 震 -> 巽 -> 乾 -> 离** 访问会让画面更清晰、干扰更少，但不再阻止其他顺序。

| 卦门 | 劫 | 关键操作 |
| --- | --- | --- |
| 坎 | 纸人劫 | 水面有抓手光标；舀水后右键纸人选择。 |
| 坤 | 契约劫 | 双击契纸空白处，再写入陈守一、为救女与签约日期。 |
| 震 | 昆仑劫 | 点开小耳朵图标，在频谱峰值读取并质询天规。 |
| 巽 | 烛龙劫 | 追随风中移动的鳞光；回复会改变烛龙态度。 |
| 乾 | 问情劫 | 先将五枚符箓拖至归元子前，再以卦象表达立场。 |
| 离 | 归元劫 | 收齐符箓、点炉；炉壁的太极可用来走超越路线。 |

### 生门、死门与终局

- 在丹炉双击，可发现生、死两门。生门道观展示归元子年轻时与“情神”的线索；死门有素心迟来的信和断笔。
- 炉壁右键可看见情卦由“天地否”“水火既济”改到“火在水上”的过程。
- 离火归元后，四个没有文字的符号对应：焚毁、刻入、保留曲线、空白超脱。选择后直接结算，不需要重跑六门。
- 已读玄澈手记、素心信，并满足隐藏收集时，结局会多出“道是他们的故事，也是我的故事”的回声。
